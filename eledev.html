eLearning Development

Fill the blanks

Esta actividad busca que los usuarios tengan una mayor interacción con la actividad, ya que no es una actividad de "scroll". Para esta actividad se utilizaron etiquetas de HTML (input) y variables condicionantes de JavaScript.

Match the columns

Este ejercicio es muy divertido, pero no sólo eso, sino que es de mucha utilidad para el desarrollo abstracto de conceptos. Además, se desarrolló con script y etiquetas tipo input que permite que los estudiantes tengan retroalimentación cuando respondan correcta o incorrectamente.

Reinforcement test

Este quiz funge como un objeto de aprendizaje de refuerzo, ya que por la tecnología empleada para su desarrollo busca que los usuarios obtengan retroalimentación en el momento de cuáles respuestas son correcas o incorrectos y, además, proporciona tabla de porcentaje que le permite saber cuál es el avance que ha tenido en el quiz.
